# **BeautyMe**
BeautyMe is a mobile app built using Kotlin. In application you can browse different cosmetics.

## **Features**
- Display cosmetics divided into three categories: Perfumes, Make-up and Body
- Display product details
- Add product to cart
- Delete products from cart
- Add product to favorites
- Delete products from favorites
- Manage products:
  - Add new product to shop
  - Delete product from shop
  - Edit product
- Display interactive advertisement with furniture

## **General informations**
- Language: **Kotlin**
- Database: **Firebase Realtime database**
- IDE: **Android Studio**
